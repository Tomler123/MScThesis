```python
def flatten_dict(d: dict, sep: str = ".") -> dict:
    """Flatten a nested dictionary by joining keys with `sep`."""
    result = {}

    for key, value in d.items():
        if isinstance(value, dict):
            # Recursively flatten the nested dict
            nested = flatten_dict(value, sep)
            for nested_key, nested_value in nested.items():
                result[key + sep + nested_key] = nested_value
            # Handle empty dict case: {"a": {}} should become {"a": {}}
            if not nested:
                result[key] = {}
        else:
            # Non-dict values are kept as-is
            result[key] = value

    return result
```

The recursion distinguishes between **leaf values** (non-dict values) and **non-leaf values** (nested dicts) by checking `isinstance(value, dict)`. When a dict is encountered, the function recursively flattens it, prepending the current key to each resulting key with the separator. For **leaf values** (strings, ints, lists, None, etc.), the function simply copies them as-is without further recursion. For the **empty-dict edge case** (`{"a": {}}`), after recursively flattening an empty dict (which returns an empty dict `{}`), the code checks `if not nested` and preserves the key with its empty dict value, ensuring `{"a": {}}` correctly maps to `{"a": {}}`.
