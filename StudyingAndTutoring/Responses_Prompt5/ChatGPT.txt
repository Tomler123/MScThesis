## 1. Key Ideas

**Simple harmonic motion (SHM)** is a type of oscillatory motion in which an object moves back and forth about an equilibrium position, and the restoring force is directly proportional to its displacement from equilibrium and points toward equilibrium.

The defining force law is:

[
F = -kx
]

where:

* (k) is the spring constant,
* (x) is the displacement from equilibrium,
* the minus sign shows the force points opposite to the displacement.

Using Newton’s second law, (F = m\ddot{x}), we get:

[
m\ddot{x} = -kx
]

or equivalently,

[
\ddot{x} + \frac{k}{m}x = 0
]

This is the differential equation for SHM.

Its general solution is:

[
x(t) = A\cos(\omega t + \phi)
]

where

[
\omega = \sqrt{\frac{k}{m}}
]

is the **angular frequency**.

The main quantities are:

* **Amplitude (A):** the maximum displacement from equilibrium.
* **Angular frequency (\omega):** how rapidly the oscillation proceeds, measured in rad/s.
* **Period (T):** the time for one complete oscillation:

[
T = \frac{2\pi}{\omega}
]

* **Phase constant (\phi):** determines the starting position and direction of motion at (t=0).

In SHM, the motion repeats regularly, and the acceleration is always proportional to displacement and directed toward equilibrium:

[
a = \ddot{x} = -\omega^2 x
]

## 2. Worked Example

A block has:

[
m = 0.5\ \text{kg}, \qquad k = 8\ \text{N/m}
]

It is pulled (0.1\ \text{m}) from equilibrium and released from rest.

We want:

1. angular frequency,
2. period,
3. explicit equation (x(t)).

### Step 1: Find the angular frequency

Use

[
\omega = \sqrt{\frac{k}{m}}
]

Substitute the values:

[
\omega = \sqrt{\frac{8}{0.5}} = \sqrt{16} = 4\ \text{rad/s}
]

So,

[
\boxed{\omega = 4\ \text{rad/s}}
]

### Step 2: Find the period

Use

[
T = \frac{2\pi}{\omega}
]

Substitute (\omega = 4):

[
T = \frac{2\pi}{4} = \frac{\pi}{2}\ \text{s}
]

Numerically,

[
T \approx 1.57\ \text{s}
]

So,

[
\boxed{T = \frac{\pi}{2}\ \text{s} \approx 1.57\ \text{s}}
]

### Step 3: Write the explicit equation (x(t))

The general form is

[
x(t) = A\cos(\omega t + \phi)
]

The block is pulled (0.1\ \text{m}) and released from rest.

That means at (t=0):

[
x(0) = 0.1\ \text{m}, \qquad v(0)=0
]

Since it starts at maximum positive displacement, the amplitude is

[
A = 0.1\ \text{m}
]

and the cosine form with zero phase fits directly:

[
\phi = 0
]

Thus,

[
x(t) = 0.1\cos(4t)
]

So the explicit equation is:

[
\boxed{x(t) = 0.1\cos(4t)}
]

## 3. Practice Exercises (with solutions)

### Exercise 1

A mass (m = 2\ \text{kg}) is attached to a spring with spring constant (k = 18\ \text{N/m}). It is displaced (0.2\ \text{m}) to the right and released from rest. Find:

1. the angular frequency,
2. the period,
3. the equation of motion.

#### Solution

First compute the angular frequency:

[
\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{18}{2}} = \sqrt{9} = 3\ \text{rad/s}
]

So,

[
\boxed{\omega = 3\ \text{rad/s}}
]

Now find the period:

[
T = \frac{2\pi}{\omega} = \frac{2\pi}{3}\ \text{s}
]

So,

[
\boxed{T = \frac{2\pi}{3}\ \text{s}}
]

The object is released from rest at maximum displacement (x=0.2\ \text{m}), so:

[
A=0.2,\qquad \phi=0
]

Therefore,

[
\boxed{x(t)=0.2\cos(3t)}
]

---

### Exercise 2

A mass (m = 1\ \text{kg}) is attached to a spring with spring constant (k = 25\ \text{N/m}). At (t=0), the mass passes through equilibrium with speed (1.5\ \text{m/s}) in the positive direction. Find:

1. the angular frequency,
2. the amplitude,
3. one possible equation of motion.

#### Solution

First compute the angular frequency:

[
\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{25}{1}} = 5\ \text{rad/s}
]

So,

[
\boxed{\omega = 5\ \text{rad/s}}
]

Now use the fact that maximum speed in SHM is:

[
v_{\max} = A\omega
]

Since the object passes through equilibrium, its speed there is maximum, so:

[
1.5 = A(5)
]

Thus,

[
A = \frac{1.5}{5} = 0.3\ \text{m}
]

So,

[
\boxed{A = 0.3\ \text{m}}
]

Now find an equation. Since the mass is at equilibrium at (t=0), we need:

[
x(0)=0
]

and since it is moving in the positive direction, a sine function works naturally:

[
x(t)=A\sin(\omega t)
]

Substitute (A=0.3) and (\omega=5):

[
\boxed{x(t)=0.3\sin(5t)}
]

This satisfies:

* (x(0)=0),
* (v(t)=1.5\cos(5t)), so (v(0)=1.5>0).

So this is a correct equation of motion.
