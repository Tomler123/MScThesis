Below is a brute-force function that groups anagrams from a list of strings. It works correctly but is very slow for large inputs. Rewrite it to be efficient.

```python
def group_anagrams_brute(words: list[str]) -> list[list[str]]:
    """Group words that are anagrams of each other (SLOW — O(n^2 * k))."""
    used = [False] * len(words)
    groups = []
    for i in range(len(words)):
        if used[i]:
            continue
        group = [words[i]]
        used[i] = True
        for j in range(i + 1, len(words)):
            if not used[j] and sorted(words[i]) == sorted(words[j]):
                group.append(words[j])
                used[j] = True
        groups.append(group)
    return groups
```
